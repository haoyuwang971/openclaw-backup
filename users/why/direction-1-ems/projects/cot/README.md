## 🎯 项目目标

实现一个光储柴微网系统的EMS能量管理策略，主要特点：
- **不做峰谷套利**，专注绿电消纳与备电
- **严格执行防逆流**，不允许向市电倒送电
- **双簇架构**，支持簇间SOC均衡
- **柴发冷备份**，根据条件启停

## 📋 技术栈

- **拓扑**: 交流耦合，双簇并联
- **控制**: 状态机切换 + 功率调度
- **均衡**: 簇间EMU均衡 + 簇内BMS均衡
- **切换**: STS快速并离网切换 + ATS市电/柴发切换
