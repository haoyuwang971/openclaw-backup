# 项目Cot - EMS定制策略分析

## 📋 项目概述

| 属性 | 内容 |
|------|------|
| **项目名称** | Cot |
| **系统类型** | 光储柴微网（交流耦合） |
| **核心目标** | 绿电消纳、弱网区备电、防逆流/需量控制 |
| **不做** | 市电峰谷套利 |

---

## 🔌 系统拓扑

### 双簇架构

```
┌─────────────────────────────────────────────────────────────┐
│                         交流母线                              │
├──────────────────┬──────────────────────────────────────────┤
│     簇1          │              簇2                          │
│  1#2#DC柜        │           3#4#DC柜                        │
│  1~4#光伏MPPT    │           5~7#光伏MPPT                    │
│  1~3#PCS         │           4~5#PCS                         │
│  (3台)           │           (2台)                           │
│  主机: 1#PCS     │           主机: 4#PCS                     │
└──────────────────┴──────────────────────────────────────────┘
                              │
                    ┌─────────┴─────────┐
                    │    STS/ATS切换     │
                    │  市电/柴发接入     │
                    └─────────┬─────────┘
                              │
                         负载侧
```

### 关键设备
- **STS**: 静态转换开关（并离网快速切换）
- **ATS**: 自动转换开关（市电/柴发切换）
- **EMU**: 能量管理单元（簇间SOC均衡）
- **3级BMS**: 电池管理系统（簇内SOC均衡）
- **PCS**: 储能变流器（1~5#，并网时电流源，离网时电压源）

## 🔄 运行模式与切换逻辑

### 模式切换状态机

```
                    ┌─────────────────┐
                    │   市电并网模式   │
                    │  (绿电消纳/备电) │
                    └────────┬────────┘
                             │ 市电故障
                             ▼
                    ┌─────────────────┐
         ┌─────────│   离网电池供电   │◄────────┐
         │         │  (PCS为电压源)   │         │
         │         └────────┬────────┘         │
         │                  │                  │
   柴发停机/SOC充足         │ SOC不足/负载大   │ 柴发启动
         │                  ▼                  │
         │         ┌─────────────────┐         │
         └────────►│   离网并柴油机   ├─────────┘
                   │  (柴发为电压源)  │
                   │   PCS为电流源    │
                   └────────┬────────┘
                            │ 市电恢复
                            ▼
                   ┌─────────────────┐
                   │   市电并网模式   │
                   └─────────────────┘
```

### 切换过程详解

| 切换过程 | 触发条件 | 动作序列 | 关键要点 |
|---------|---------|---------|---------|
| **并网→离网** | STS检测到市电不同期 | STS快速切断→离网电池供电 | 1~5#PCS自动变电压源，保障不间断供电 |
| **离网→并网** | ATS检测到市电同期 | ATS闭合→STS检测同期→并入 | 1~5#PCS变电流源，功率指令下发给主机 |
| **离网→并柴发** | SOC不足/负载大 | EMU启动柴发→ATS闭合→STS同期并入 | 柴发为主要电压源，PCS为电流源 |
| **并柴发→离网** | SOC充足/柴发轻载 | 停止柴发→STS切断→离网电池供电 | PCS变回电压源 |
| **并柴发→并网** | 市电恢复 | ATS优先切市电→STS同期并入 | 柴发侧先断开，再并市电 |

## ⚡ 核心控制策略

### 1. 防逆流与需量控制
- **严格执行**: 不允许向市电倒送电
- **约束条件**: 变压器端功率限制 Plimit
- **控制手段**: 光伏限功 + PCS功率调节

### 2. 簇间SOC均衡（并网充电时）

**场景**: 充电需求大，受市电需量约束

**均衡公式**:
```
簇1可用功率 = Plimit × (ΔSOC₁ / (ΔSOC₁ + ΔSOC₂))
簇2可用功率 = Plimit × (ΔSOC₂ / (ΔSOC₁ + ΔSOC₂))

其中:
ΔSOC₁ = 簇1距离充电DOD上限的差值
ΔSOC₂ = 簇2距离充电DOD上限的差值
```

**示例**: 
- Plimit = 100kW
- 簇1还差30%到DOD上限
- 簇2还差60%到DOD上限
- 簇1功率 = 100 × (30/90) = 33.3kW
- 簇2功率 = 100 × (60/90) = 66.7kW

### 3. 光伏限功策略

| 簇 | 光伏MPPT | 限功公式 |
|---|----------|---------|
| 簇1 | 1~4# | P_limit = P_电池需求 + P_PCS实时(正放负充) - 保护余量 |
| 簇2 | 5~7# | P_limit = P_电池需求 + P_PCS实时(正放负充) - 保护余量 |

**要求**: 功率能快速限制，跟随PCS功率输出变化

### 4. 并网状态下PCS功率设置

**根据SOC阈值判断**:

| SOC范围 | PCS策略 | 负载能量来源 | 电池能量来源 | 场景 |
|---------|---------|-------------|-------------|------|
| DOD < SOC < 阈值1 | 负载跟随(min(负载,光伏)) | 光伏 | 光伏 | 光照强 |
| 阈值1 < SOC < 阈值2 | 不充不放 | 市电 | 光伏 | 光照弱 |
| 阈值2 < SOC < DOD | 市电充电 | 市电 | 市电 | 夜间/无光 |

**阈值设置示例**:
- 夜间/阴天: 阈值1=94%, 阈值2=90%
- 早晚: 阈值1=80%, 阈值2=70%
- 正午: 阈值1=70%, 阈值2=60%

### 5. 柴发启停逻辑

**启动条件** (同时满足):
- SOCmin < 启动柴发SOC
- 市电未恢复
- 光伏不足

**停止条件** (满足其一):
- 市电恢复，ATS切至市电
- 电池SOC达到停止柴发阈值

**并柴发时PCS设置**: PCS功率 = 0（不通过柴发给电池充电）

### 6. 滞环控制

所有SOC判断需带滞环，防止实时数据抖动导致频繁控制:
- "实时值 > 设定值" 实际为 "实时值 > 设定值 + **2%**"
- "实时值 < 设定值" 实际为 "实时值 < 设定值 - **2%**"

**滞环值可配置**。

## 🔧 技术要点总结

1. **PCS模式切换**: 并网时电流源（控功率），离网时电压源（建压频）
2. **功率均分**: 
   - 并网时：1~3#PCS功率下发1#主机，4~5#PCS功率下发4#主机
   - 离网时：1~5#PCS作为整体，不控功率，主机为1#PCS
3. **光伏均分**: 1~4#MPPT由1#主机均分，5~7#MPPT由5#主机均分
4. **DOD防护**: 无论什么场景，电池DOD防护、电芯防护按标准策略执行
5. **冷备份**: 柴发非热备份，需启动时间
