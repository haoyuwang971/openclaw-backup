## 📝 使用建议

做 **EMS控制策略设计** 时，建议交叉阅读：
1. **GB/T 42726-2023**（监控功能要求）
2. **GB/T 36558-2023**（系统级功率控制与响应指标）
3. **GB/T 36547-2024**（并网调频、惯量、故障穿越要求）
4. **GB/T 36274-2018**（微网多模式切换，如果Cot涉微网）
5. **GB/T 34131-2023**（BMS数据接口）
6. **GB/T 34120-2023**（PCS控制接口与响应能力）
