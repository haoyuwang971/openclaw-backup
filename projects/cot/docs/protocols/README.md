#### 2.1.8 故障复位规则

| 故障类型 | 复位方式 |
|---------|---------|
| 交流主继电器粘连 | **禁止上电，联系厂家** |
| 直流/交流主继电器开路 | **禁止上电，联系厂家** |
| 硬件故障字1/2（除功率模块过温） | **下电复位** |
| 电池电压/温度类故障 | 恢复后**自动复位** |
| 其余故障 | 关机状态下发**故障复位指令(0x1400 Bit15=1)** |

#### 2.1.9 PCS控制流程

**充电流程**
1. 开机命令 (0x0291 = 1)
2. 写负功率到 0x0D57 设定充电功率
3. 充满后关机命令 (0x0291 = 0)

**放电流程**
1. 开机命令 (0x0291 = 1)
2. 写正功率到 0x0D57 设定放电功率
3. 放完后关机命令 (0x0291 = 0)

**Cot项目PCS开机序列（离网模式）**
1. 设置 PCS 1# 功率 = 0，等待建立离网电压
2. 依次启动 PCS 2# → 3# → 4# → 5#

**Cot项目PCS关机序列**
1. 设置 PCS 2#~5# 功率 = 0，依次停机
2. 最后停止 PCS 1# (**VF主机必须最后退出**)

**功率分配策略**
- 簇1 (PCS 1#~3#): 主机 PCS 1# 控制总功率，均分给 3 台
- 簇2 (PCS 4#~5#): 主机 PCS 4# 控制总功率，均分给 2 台

#### 2.3.6 Cot项目MPPT控制逻辑

**功率限值控制（EMS直接控制每台MPPT）**

⚠️ **重要变更**: EMS直接对每台MPPT独立设置功率百分比，不再通过主从机分配。

**控制架构**
- **簇1** (MPPT 1#~4#): 总功率边界 [0, 400kW]
  - EMS分别对1#、2#、3#、4#写 46110 功率百分比
  - 各台功率百分比 = 该台分配功率 / 该台基准功率 × 100%
  
- **簇2** (MPPT 5#~7#): 总功率边界 [0, 300kW]
  - EMS分别对5#、6#、7#写 46110 功率百分比
  - 各台功率百分比 = 该台分配功率 / 该台基准功率 × 100%

**功率分配计算**
1. 计算簇总PV功率限制：P_limit_total = 95% × (簇电池充电需求 - 簇PCS实时功率)
2. EMS根据各MPPT状态（在线/故障/功率余量）将P_limit_total分配给各台
3. 各台独立设置46110功率百分比

**重要约束**
- MPPT **只能单向控制，不能放电**（功率≥0）
- 限功率通过百分比实现，而非直接功率设定值
- 固化参数时，60/120kW机型**必须关机**，320kW无需关机
- **EMS需维护7台MPPT各自的功率限制状态**

**EMS控制流程**
1. 读取 45200 状态字，确认bit8=1（允许启机）
2. 写 46001=1 开机
3. 根据功率分配策略，分别写各台46110功率百分比
4. 读取各台45217输入侧总功率，确认实际输出功率
5. 停机时写 46001=0 关机

**与旧方案对比**
| 项目 | 旧方案（主机分配） | 新方案（EMS直接控制） |
|------|-------------------|----------------------|
| 控制主体 | MPPT 1#/5#作为主控分配 | EMS直接控制每台 |
| 设置对象 | 仅设置主控MPPT | 分别设置7台MPPT |
| 通信负载 | 低 | 高（7路独立控制） |
| 灵活性 | 受限于主控算法 | EMS灵活分配策略 |

#### 2.6.7 Cot项目液冷机控制逻辑

**核心监控参数**
1. **出水/回水温度** (0xE700/0xE701) - 热管理核心指标
2. **告警最高等级** (0xE715) - 故障分级处理依据
3. **心跳** (0xE71C) - 通信状态监控
4. **开关机状态** (0xE731) - 设备状态确认

**EMS控制策略**
1. **温度控制**:
   - 根据BAM电池温度，计算目标出水温度
   - 写0xE600制冷点 / 0xE601加热点
   - 建议制冷点25.0℃，加热点15.0℃（根据实际调整）

2. **运行模式**:
   - 正常: 0x0401=4（全自动）
   - 维护: 0x0401=1（自循环）
   - 停机: 0x0401=0（停止）

3. **启停控制**:
   - 开机: 0x0400=1
   - 关机: 0x0400=0

4. **告警处理**:
   - 0xE715 ≥ 1: EMS触发告警通知
   - 0xE715 ≥ 2: EMS评估是否需要降功率或停机
   - 0xE715 = 3: EMS建议紧急停机

**重要约束**
- 液冷机与DC柜一一对应（1#~4#）
- 温度设定需根据BAM电池温度反馈动态调整
- 心跳(0xE71C)需定期读取监控通信状态
