## 功率方向定义修正 (2026-04-10 18:45)

**用户指出**: 功率方向定义混乱，必须统一标准

### 统一功率方向定义（用户标准）

| 变量 | 正方向 | 负方向 | 说明 |
|------|--------|--------|------|
| **P_bat** | 充电（功率流入电池） | 放电（功率流出电池） | ⚠️ 注意：与用户之前说的相反，以本次为准 |
| **P_pcs** | 放电（DC→AC，电流从直流流向交流） | 充电（AC→DC） | 交流侧功率定义 |
| **P_pv** | 发电（总是正值） | - | 光伏输出功率 |

### 直流侧功率平衡方程

```
P_bat = P_pv - P_pcs/η

其中:
- P_bat > 0: 电池充电
- P_bat < 0: 电池放电
- P_pcs > 0: PCS放电(逆变DC→AC)
- P_pcs < 0: PCS充电(整流AC→DC)
- P_pv > 0: 光伏发电
- η: PCS效率(约98%)
```

### 场景验证

**场景1: 纯光伏充电 (PV=100kW, 负载=0, PCS=0)**
```
P_bat = 100 - 0/η = 100 > 0 → 电池充电 ✅
```

**场景2: 光伏+放电供负载 (PV=50kW, 负载=100kW)**
```
PCS放电50kW到交流侧 → P_pcs = 50
P_bat = 50 - 50/η = 50 - 51 ≈ -1 < 0 → 电池放电 ✅

实际:
- 光伏提供50kW
- PCS从母线取电51kW(η=0.98)，输出50kW给负载
- 电池放电1kW补充？不对...

等等，让我重新算:

PV=50, 负载=100
PCS需要提供100-50=50kW给负载
PCS从直流母线取电 = 50/η = 51kW
电池放电 = 51 - 50 = 1kW

用公式: P_bat = P_pv - P_pcs/η = 50 - 50/0.98 = 50 - 51 = -1 < 0 ✅
```

```
P_bat = P_clu/η - P_pv  ❌ 完全颠倒
```

**正确公式**:

```
P_bat = P_pv - P_clu/η  ✅
```
